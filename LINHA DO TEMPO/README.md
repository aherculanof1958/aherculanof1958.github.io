# Linha do Tempo Interativa

Projeto de linha do tempo interativa desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades
- Adicionar novos eventos
- Navegar entre anos
- Design responsivo
- Armazenamento local (localStorage)

## Como usar
1. Acesse [https://aherculanof1958.github.io](https://aherculanof1958.github.io)
2. Clique em "Adicionar Evento" para criar novos eventos
3. Use os botões "Anterior" e "Próximo" para navegar entre anos

## Tecnologias
- HTML5
- CSS3 (com Flexbox)
- JavaScript (ES6+)
- Font Awesome para ícones
- Google Fonts
